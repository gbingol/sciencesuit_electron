Run following commands from the main project folder: 

1) npm install --save-dev @electron-forge/cli
2) npx electron-forge import
3) npm run make


After the out directory is created:

1) copy dll file where the .node file is
2) The system must have VC++ runtime: https://learn.microsoft.com/en-US/cpp/windows/latest-supported-vc-redist?view=msvc-170
must be downloaded and installed.
